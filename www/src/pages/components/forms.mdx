# Forms

```jsx react-live
<form>
  <FormGroup id="id" type="text" label="Id" />
  <FormGroup id="password" type="password" label="Password" />
  <div>
    <Checkbox id="checkbox" label="로그인 저장" labelWith />
  </div>
  <Button type="submit" color="primary" text="로그인" />
</form>
```
